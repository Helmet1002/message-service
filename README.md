backend system for a simple music streaming application using Java.
Tech Stack: Java,Spring Boot, H2(for dev testing),Postman,Maven,JPA
Core Features:
1. User Management
o Register/Login
o View profile
2. Song Management
o Add (admin only), list, and search songs (by title, artist, genre)
3. Playlist Management
o Create playlists
o Add/remove songs
o View user playlists
4. Playback Simulation
o Play, pause, resume, stop song
o Show current playing song for a user
